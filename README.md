# FAYNX — Free HD & 4K Wallpapers PWA
### Redesigned Edition — Luxury Editorial Dark Magazine Aesthetic

A fast, SEO-optimised Progressive Web App powered by the Unsplash API and Firebase, rebuilt with a high-end editorial design system.

## Stack

- **Frontend**: Vanilla JS (ES Modules), CSS custom properties, no build step
- **Fonts**: Cormorant Garamond + DM Sans via Google Fonts
- **Images**: Unsplash API → Imgix CDN (WebP, quality=80)
- **Auth**: Firebase Authentication (Email + Google)
- **Database**: Firebase Firestore (saves, collections, downloads, prefs)
- **PWA**: Service Worker v4, Web App Manifest v2
- **SEO Engine**: Node.js generator (`generate.js`) — 960+ pages

## Setup

### 1. Install Node (for generate.js — no npm dependencies needed)
```bash
node --version   # requires Node 18+
```

### 2. Run the SEO generator
```bash
node generate.js
```
Outputs:
- `wallpaper/*.html` — 960+ individual wallpaper pages
- `category/*.html` — 6 category landing pages
- `sitemap.xml`, `sitemap-main.xml`, `sitemap-images.xml`
- `robots.txt`

### 3. Configure Firebase (REQUIRED before deploy)
In `app.js`, replace the Firebase config object at the top with your project config from the Firebase Console → Project Settings.

**⚠ Security:** Before going to production, move the Firebase config and Unsplash key to environment variables via a serverless proxy. See Step 1 of the Transformation Roadmap for complete code.

### 4. Configure Unsplash (REQUIRED)
Replace `KEY` in `app.js` with your Unsplash Access Key from [unsplash.com/developers](https://unsplash.com/developers).

Free tier: **50 requests/hour**. Apply for Production Access for 5,000/hour.

### 5. Deploy
```bash
# Netlify (recommended — supports Edge Functions for API proxy)
netlify deploy --prod

# Or any static host: Vercel, Cloudflare Pages, GitHub Pages
```

## SEO Architecture

### Wallpaper Pages (960+)
Each page includes:
- Self-referencing `<link rel="canonical">`
- `ImageObject` + `WebPage` + `BreadcrumbList` JSON-LD
- Keyword-rich alt text: `"${kw} — free HD 4K wallpaper download"`
- 150–300 words of **unique** content via content variation engine
- Related wallpapers section (scored by keyword + cluster overlap)
- 6 download size options (HD → 4K → Mobile → iPhone → Android)
- WebP image delivery via Unsplash Imgix URL params

### Category Pages (6)
- `CollectionPage` + `ItemList` + `BreadcrumbList` JSON-LD
- 48-image grid with keyword filter pills
- 200+ words of unique SEO copy per category
- Cross-category internal linking

### Sitemaps
- `sitemap.xml` — index pointing to two child sitemaps
- `sitemap-main.xml` — all URLs with priority + changefreq
- `sitemap-images.xml` — `image:image` entries for Google Images indexing
